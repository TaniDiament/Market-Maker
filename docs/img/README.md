# Exported diagram images

Source `.drawio` files live in `/diagrams/`. To refresh the images referenced from `/README.md`:

1. Open the source file in draw.io desktop (or app.diagrams.net).
2. `File → Export As → JPEG…` (or PNG).
3. Save into this directory using the matching name, e.g.:
   - `diagrams/marketmaker.drawio` → `docs/img/architecture.jpeg`
